# 无界学城穿梭巴士

深圳大学城（南山区）免费校园摆渡车，服务清华、北大、哈工大三所院校。三条线路均为环形，每班约 40 分钟跑完全程。

## 获取实时数据

```bash
python3 "{baseDir}/scripts/fetch.py" [线路]
```

线路参数：`1`、`2c`（顺时针）、`2a`（逆时针），不填则查询全部。脚本无外部依赖，自动匿名登录，输出 JSON，每辆车一条：

```json
{
  "plate": "深D000047",
  "online": true,
  "current_station": "清华教学楼C栋(去程)",
  "station_sort": 8,
  "direction": "西",
  "speed_kmh": 9.0,
  "gps_time": "2026-04-09 18:04:35",
  "line": "无界学城一号线",
  "line_key": "1"
}
```

列表为空 = 当前无车运行（非运营时段或班次间隙）。

## 估算到站时间

每站约 2–3 分钟。剩余站数 = (目标 sort − 当前 sort + 总站数) % 总站数，乘以 2.5 分钟即可。
